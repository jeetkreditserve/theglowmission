# The Glow Mission UI Reference

Primary references copied into the project:

- `apps/frontend/public/reference/glow-mission-post-1.png`
- `apps/frontend/public/reference/glow-mission-logo-3d.png`
- `docs/design/reference/style-reference.png`

Visual system:

- Soft ivory `#FFF7F0`, warm cream `#F6EEE4`, champagne gold `#D9B88C`, soft taupe `#CDB8A9`, nude beige `#E6D6C6`, espresso `#2B2623`.
- Display typography should feel Cinzel-like: refined, uppercase-capable, generous spacing only for labels/nav.
- Body typography should feel Montserrat/Lato-like: clean, readable, restrained.
- Public site should echo `Glow Mission Post 1.png`: warm natural skincare objects, bright ivory surface, gold logo/copy, citrus/honey/cucumber/cream details, and a calm mission-forward tone.
- Admin CMS should use the same palette but stay operational: dense rows, sidebar navigation, compact stats, and restrained controls.

Copy anchors:

- Brand: `The Glow Mission`
- Tagline: `Glow, the natural way.`
- Essence: `Soft. Elegant. Timeless. Made to make you glow.`
- Launch tone: `Skin that glows. Confidence that shows.`
